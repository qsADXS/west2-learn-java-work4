<a id="schema_user"></a>
<a id="tocSuser"></a>
<a id="tocsuser"></a>

```json
{
  "username": "xiaoming",
  "userId": 1,
  "password": "password"
}

```

### 属性

|名称|类型|必选|约束|中文名|说明|
|---|---|---|---|---|---|
|username|string|true|none|用户名|单一用户名|
|userId|integer|true|none|用户id|none|
|password|string|false|none|密码|密码不会传递到前端（只是暂时不知道怎么把这个单独去掉）|
